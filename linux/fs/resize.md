# Расширение корневой партиции в Linux

## Пересоздание партиции

Для расширения корневой партиции `/dev/sda1` на свободное пространство выполните следующие шаги:

#### 1. Проверка текущей разметки
```bash
lsblk
df -h
parted /dev/sda print free
```

#### 2. Удаление существующей партиции (данные не потеряются)
```bash
parted /dev/sda
(parted) rm 1
(parted) print free
```

#### 3. Создание новой расширенной партиции
```bash
(parted) mkpart primary xfs 1049kB 644GB
(parted) set 1 boot on
(parted) print
(parted) quit
```

#### 4. Расширение файловой системы
```bash
xfs_growfs /
```

#### 5. Проверка результатов
```bash
df -h
lsblk
```

### Важные замечания:
1. **Тип таблицы разделов**: У вас используется MBR (msdos), которая имеет ограничение в 2TB
2. **Резервное копирование**: Перед операцией рекомендуется сделать бэкап важных данных
3. **XFS файловая система**: Для расширения используется `xfs_growfs`, а не `resize2fs`

Если вам нужно преобразовать таблицу разделов в GPT для поддержки больших объемов, процесс будет сложнее и потребует переустановки системы.
